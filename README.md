
# 🚀 HackStudy

> Hack your studies. Learn smarter.

HackStudy is a modern academic resource platform designed to help college students access **sessional papers, previous year questions, and important study materials** in one place.

---

## 🌐 Live Demo

👉 https://roshanworld.github.io/HackStudy/

---

## 📌 Features

- 📄 Access Sessional Papers (Sessional 1, Sessional 2, PUT)
- 📚 Organized subject-wise content
- 🔗 Google Drive integration for paper access
- 🌙 Modern dark UI (Black + Orange theme)
- ⚡ Fast and simple navigation
- 📱 Responsive design

---

## 🧠 Problem It Solves

Students often struggle to find:
- Previous sessional papers  
- Important repeated questions  
- Centralized study resources  

HackStudy provides a **single platform** for all academic materials.
